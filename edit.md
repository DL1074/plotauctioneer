Capture Flow (Already Done)
Player selects region with axe
/plotconfirm MyHouse
Build → Schematic file + Plot item (no price yet)
Player can either:
List on auction house plugin
Keep to paste later
OR create a shop (new)


Shop Creation Flow (New System)
Player selects region with axe
Instead of /plotconfirm, they do /plotshop create
System gives them a custom shop block (like a special sign/block)
Player places the shop block
Player types price in chat (e.g., 5000)
Build saves data on the sign, including price, and the 2 points the player selected for the region
Shop block shows: price, name, "Right-click to buy"

Browsing & Purchase Flow
Buyer walks through the actual build (not pasted)
Explores rooms, sees furnishings, checks design
Right-clicks the shop block
Build → Schematic file + Plot item

Plot item created and given to buyer
Money: buyer → seller
Buyer can now paste it wherever they want
